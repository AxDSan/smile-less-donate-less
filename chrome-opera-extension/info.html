Smile Less, Donate Less :)

See Your AmazonSmile Impact

This lightweight extension stops redirecting you to the annoying smile.amazon.com mirror page every time you visit Amazon. AmazonSmile is Amazon's charity program for non-profit organizations. It's annoying AF and it costs money, and it cannot be turned off.

This extension is not affiliated with, maintained, authorized, endorsed or sponsored by Amazon.com. Get the open source code for the original project.
